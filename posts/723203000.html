Ventoy 界面自定义教程

Ventoy 也能豪堪？教你自定义 Ventoy 主题！

软硬件需求清单

类别 名称 / 说明
浏览器 任意现代浏览器（推荐 Chromium 108+）
压缩软件 Bandizip / WinRAR
Ventoy 本体 官网 Download . Ventoy
图像处理 Adobe Photoshop / PowerPoint / MasterGO
虚拟机 VMware Workstation Pro（123云盘免登录下载 或博通官网注册）
操作系统 Windows 10 / 11
网络 可正常联网
U 盘 ≥ 16 GB，用于制作 Ventoy 启动盘

正式步骤

1. 获取 GRUB 主题

  1. 打开浏览器，搜索 “Grub 主题” 或直接访问 Gnome-look.org › GRUB Themes。
  2. 以 Elegant-forest 为例，进入其 详情页：
    • 选择 Files → Elegant-forest-window-grub-themes.tar.xz（≈ 12 MB）。
  3. 下载后用 Bandizip / WinRAR 解压，选择与自己分辨率匹配的文件夹（1920×1080 或 1280×720）。

2. 自定义背景

  1. 进入解压后的主题文件夹，仅保留主题主目录，其余删除。
  2. 用 PowerPoint（或 PS）新建空白演示文稿：
    • 版式 → 空白。
    • 把 background.jpg 拖入作为背景。
  3. 设计思路（可替换为个人风格）：
    • 插入 圆角矩形 覆盖中间区域，透明度 40%。
    • 复制一个对齐，再插入第三个圆角矩形框出中间不透明部分。
    • 删除原 background.jpg，替换为自选背景并设置高斯模糊（半径 100）。
    • 新建白色全屏矩形，透明度 10%–20%，置于最上层。
    • 导入 info.jpg 做布局参考，重绘或替换图标、文字、Logo 等。
  4. 完成后导出为 background.jpg（1920×1080）。

作者已提供 PPT 模板，可在此基础上快速修改：
Ventoy主题.pptx 提取码：gary

3. 修改 theme.txt

  1. 打开主题根目录下的 theme.txt。
  2. 如无 logo.png，删除对应行；如有，请保留。
  3. 字体（可选）：将所有 .pf2 字体文件路径写入 theme.txt 的 fonts 字段。

4. 部署到 Ventoy

  1. 将 Ventoy 写入 U 盘（建议 GPT + UEFI）。
  2. 把改好的主题文件夹整体复制到U盘
  3. 运行 Ventoy 安装目录下的 VentoyPlugson.exe：
  • 选择 U 盘后点击 启动，浏览器会自动打开本地配置页面。
  • 左侧 → 主题插件 → theme-file → 新增
    填写主题文件夹内 theme.txt 的绝对路径（去掉系统自带引号）。
  • gfxmode → 分辨率 建议 1920x1080（兼容性最佳）。
  • fonts → 新增 依次添加所有 .pf2 字体文件。
  • 菜单类型插件 → menu_class → 新增
    通过镜像文件名关键字匹配图标（可选，仅美化）。
  1. 完成后点击 停止管理进程。

效果验证

VMware 测试

  1. 新建虚拟机 → 自定义 → UEFI + 安全启动。
  2. 虚拟机设置 → 电源 → 启动时进入固件。
  3. 右下角挂载 Ventoy U 盘 → Reset → ESC → 选 EFI USB Device。
  4. 若出现 MOK 管理界面，按提示：
  • Enroll key from disk → 选择 ENROLL_THIS_KEY_IN_MOKMANAGER.cer → YES → Reboot。
  1. 成功启动即可看到自定义主题。

实体机启动

  1. 开机瞬间按主板快捷键（常见：ESC / F2 / F7 / F8 / F12 / DEL）。
  2. 选择 UEFI: (U盘品牌/型号) 回车即可。
